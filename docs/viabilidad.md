# Viabilidad de Automatización

La aplicación My Demo App es viable para automatización debido a:

- Los flujos son repetitivos.
- La aplicación cuenta con elementos identificables.
- Existen IDs accesibles para Appium.
- Los procesos críticos requieren validación frecuente.
- Permite ejecución en Android e iOS.

## Flujos Automatizados

- Login
- Agregar productos
- Checkout

## Beneficios

- Reducción de tiempo manual
- Validación continua
- Integración con CI/CD
- Reutilización de scripts

## Riesgos

- Cambios de UI
- Selectores dinámicos
- Dependencia de emuladores/simuladores
