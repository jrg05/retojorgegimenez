# Definición de Escenarios

## Escenario 1 - Login Exitoso

Objetivo:
Validar que un usuario pueda iniciar sesión correctamente.

Prioridad:
Alta

Flujo:
1. Abrir aplicación
2. Ingresar username
3. Ingresar password
4. Presionar Login

Resultado esperado:
El usuario accede correctamente al catálogo de productos.

---

## Escenario 2 - Agregar Producto al Carrito

Objetivo:
Validar que un producto pueda agregarse al carrito.

Prioridad:
Alta

Flujo:
1. Login exitoso
2. Seleccionar producto
3. Agregar al carrito

Resultado esperado:
Producto agregado correctamente.

---

## Escenario 3 - Checkout

Objetivo:
Validar el flujo de compra.

Prioridad:
Crítica

Flujo:
1. Login
2. Agregar producto
3. Ir al carrito
4. Checkout

Resultado esperado:
Checkout completado correctamente.
